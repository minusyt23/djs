File specs:

Header:
PROPERTY            BYTES
Track Name          16
Author Name         16


Encoding specs:

For PU1:

I -> PLAY, SET

PLAY (I = 0):
                                        (IF X = 1)
    I N N N N N N N   X L L L L L L L   [L L L L L L L L]
    N = midi Note (0 - 127).
    X = eXtra length.
    L = Length before new instruction (in frames).

SET (I = 1):
    I V S - - - - -
    V = Volume.
